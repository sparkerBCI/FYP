Detailed Description

This file holds declarations for functions to communicate over the serial interface.

This includes Error, Warning, and Debug messages, which are formatted to include the runtime (HH:MM:SS.MS format). These messages can be globally enabled by setting msg_enabled to true. Debug messages require an additional setting of debug_enabled to true.
