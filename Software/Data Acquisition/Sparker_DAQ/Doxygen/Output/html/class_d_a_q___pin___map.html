typedef struct DAQ_Pin_Map::Pin_Array_t DAQ_Pin_Map::Pin_Array_t

Toggles the state of a pin given the name of a pin. This is slow, and should only be used for non-time critical applications (e.g. indicator LEDs, not trigger signals).

This struct contains information for each entry in the Pin Array.

◆ get_state()

uint8_t DAQ_Pin_Map::get_state ( Pin_ID_t  pin)

Gets the state of the pin, as stored in the pin array.

Doesn't work well for input pins, as these pins are changed by external factors. Also is more complex than it should be, as pins that changed before calling an update will need to be flipped to get the actual pin state.

Parameters
[in]pin- The Pin_ID_t of the pin to check
Returns
uint8_t - The state of the pin, as stored in the Pin_Array

◆ set_state()

uint8_t  state 
)

Changes the state of the pin in the pin array, and marks it for updating.

Note, this does not actually update the state of the physical pin, and an update must be called to actually push this change onto the pin.

Parameters
[in]pin- The Pin_ID_t of the pin to change
[in]state- The new state of the pin
