Detailed Description

This file holds the ADS1299_Module class function declarations.

All the getters and setters are here. There are also functions to send commands (of type Command_t) to the device. The functions to write raw data to the registers are private to try to limit what people can throw at the device (some register bits are reserved, and may cause undefined behaviour if they are changed).
